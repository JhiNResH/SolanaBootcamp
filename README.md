## 技術堆疊

- Solana
- Anchor Framework
- TypeScript
- Node.js

## 開發環境設定

1. 安裝 Solana CLI
2. 安裝 Anchor Framework  
3. 安裝 Node.js 和 yarn
4. 進入個別專案資料夾執行 `yarn install`
